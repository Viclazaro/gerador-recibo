<h1 align="center">🧾 Gerador de Recibo com FPDF</h1>

<p align="center">
  Projeto desenvolvido para gerar <strong>recibos em PDF</strong> utilizando <strong>PHP</strong> e a biblioteca <strong>FPDF</strong>.  
  Ideal para automatizar a criação de documentos simples de forma rápida e prática.
</p>

---

## ➢ Sobre o Projeto

O **Gerador de Recibo** permite criar recibos personalizados em formato PDF diretamente pelo navegador.

- Geração dinâmica de recibos
- Inserção de dados como nome, valor e descrição
- Formatação automática do documento
- Download do PDF pronto
